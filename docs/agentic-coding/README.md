# Agentic Coding (Yapay Zeka ile Kodlama) Serisi

Merhaba! Bu kılavuz, kodlama süreçlerini yapay zeka ajanları (AI Agents) yardımıyla nasıl inanılmaz derecede hızlandırıp otomatize ettiğimi anlattığım serinin ana merkezidir.

Agentic Coding (Otonom Kodlama), kafanızdaki fikri veya uygulamayı geleneksel yöntemlerle satır satır yazmak yerine, yapay zekaya "doğru talimatları (prompt/issue)" vererek uygulamanızı yazdırma sanatıdır. Bu sayede sadece test, demo ve MVP (Minimum Viable Product) projelerinizi değil, çok daha kompleks yapıları bile aylar yerine günler içinde inşa edebilirsiniz.

## Bu Kılavuzda Neler Var?

Bu dosyalar sıfırdan hiç bilmeyen birine göre hazırlanmıştır. Sırasıyla okuyarak siz de projelerinizi hayata geçirebilirsiniz:

1. [Temel Kavramlar ve Kurulum](./01-temel-kavramlar-ve-kurulum.md) 
   *(GitHub nedir? Agent nedir? Ücretsiz Öğrenci hesabı nasıl alınır?)*
2. [Bizim Yapı Nasıl Çalışıyor?](./02-bizim-yapi-nasil-calisiyor.md) 
   *(Projeleri nasıl Issue'lara bölüyoruz? Neden bu şekilde yapıyoruz?)*
3. [Adım Adım Kullanım Rehberi](./03-adim-adim-kullanim-rehberi.md) 
   *(Web'den veya Cep Telefonundan kopyala/yapıştır ile ajanlara görev verme)*
4. [Örnek Kullanılabilir Şablonlar](./04-kullanilabilir-ornek-sablonlar.md) 
   *(Kendi projelerinizde hemen kullanmaya başlayabileceğiniz hazır promptlar)*
5. [Sosyal Medya Video Senaryosu](./05-sosyal-medya-video-senaryosu.md) 
   *(Bu sistemi anlattığım instagram ve linkedin metin/video içerikleri)*

> Siz de bu şablonları kullanarak sadece birkaç tıklamayla (hatta bazen bilgisayarınızı bile açmadan telefondan!) projelerinizi adım adım yapay zekaya kodlatabilirsiniz.

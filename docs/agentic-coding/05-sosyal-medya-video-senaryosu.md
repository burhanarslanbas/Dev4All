# 05 - Sosyal Medya Video / İçerik Senaryosu

Instagram (Reels), YouTube (Shorts) ve LinkedIn'de paylaşıp yüksek etkileşim (ve takipçi) elde edebileceğiniz video içerik senaryosudur.

**Videonun Amacı:** İnsanlara yapay zeka ile kod yazmanın ChatGPT'den metin kopyalamaktan ibaret olmadığını; asıl vizyonun "Agent" denilen araçlara otonom şekilde iş atamak olduğunu somut olarak göstermek!

---

### Format: 60-90 Saniyelik Dikey Video (Reels/Shorts)

**[00:00 - 00:05] KANCA (Hook) - Soru ve Dikkat Çekme**
*(Ekranda Sen (Kamera Açık) - Elinde sadece akıllı telefon var, bilgisayar yok)*
**Konuşma Metni:** "Geçen gün sadece kafede telefonumda takılırken uygulamanın Şifremi Unuttum sayfasını baştan aşağı kodladım. Üstelik bilgisayarımı açmama bile gerek kalmadı. Nasıl mı?"

**[00:05 - 00:15] PROBLEM & FARKINDALIK**
**Konuşma Metni:** "Çoğu kişi ChatGPT'ye girip 'Bana kod yaz' diyor, sonra o metni alıp dosyaları düzeltmekle saatlerini harcıyor. Oysa profesyoneller 'Agentic Coding' yöntemini kullanıyor."
*(Ekranda: ChatGPT arayüzü üstüne çarpı işareti (❌) konulmuş basit bir animasyon veya görsel efekt)*

**[00:15 - 00:35] ÇÖZÜM GÖSTERİMİ (Ekran Kaydı Aşaması)**
*(Kamera Seni Değil, Telefon Ekran Kaydını Gösteriyor)*
**Konuşma Metni:** "Ajanlar(Agent) verdiğiniz dosyayı, klasörü algılar, mimari kurallarınızı anlar ve kodları kendisi yazıp yerine koyar. GitHub üzerinde önceden hazırladığım iş dosyasını (Issue'yu) açıyorum."
*(Ekran: `13-all-issue-agent-prompts.md` veya Issue içindeki prompt telefondan kopyalanıyor)*

**[00:35 - 00:45] AGENT EKRANI**
**Konuşma Metni:** "Kopyaladığım görevi GitHub Copilot (veya Cursor / Roo) Agent ekranına yapıştırıp onayla diyorum. Arke planda dosya yaratmasını, hata düzeltmesini, hatta kodu test etmesini film izler gibi izliyorum!"
*(Ekran: Agent'ın kendi kendine loglar üretmesi, yeşil onay tikleri veya dosyaları taraması hızlandırılmış(x2) şekilde gösterilir)*

**[00:45 - 00:55] SONUÇ**
**Konuşma Metni:** "Sadece test ürünlerim (MVP) ve demolarım için bunu yaparak geliştirme süremi 3-4 haftadan sadece 1-2 güne indirdim. İşin en güzel yanı da öğrenciyseniz bu premium ajanlar GitHub üzerinden tamamen ücretsiz!"
*(Ekranda: "GitHub Student" rozeti)*

**[00:55 - 01:00] EYLEME ÇAĞRI (CALL TO ACTION)**
*(Tekrar Yüzün)*
**Konuşma Metni:** "Bu yapıyı sıfırdan nasıl kuracağınızı ve telefonda kullandığım hazır 'Kopyala-Yapıştır' ajan şablonlarımı profilimdeki linke/dokümana ücretsiz koydum. Hemen girip inceleyin ve takip etmeyi unutmayın!"
